## 📖 About the Project

EuroLeaguesScore is a shorterm developped application that could help a normal football fan track his favourites.

The Web application now has only tracking possibilities. It has a favourite feature where you can add your favourite teams and players.
The system is still in development mode and it will upgrade much more.
The plan for the future is not to have only players and teams... but real live scores of Matches (entity) and also other sports.
